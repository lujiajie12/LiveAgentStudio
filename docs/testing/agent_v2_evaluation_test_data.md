# Agent v2 路由与 QA 检索评测数据集

生成日期：2026-04-18
适用范围：Router 路由准确率评测、错路由归因分析、QA 检索 Recall@5 人工评测。

## 使用说明

- 路由标签统一使用：`direct`、`qa`、`script`、`tool`、`analysis`。
- 如果系统内部使用 `analyst`，评测时把本文的 `analysis` 映射为 `analyst`。
- `模型预测路由`、`是否正确`、`错误类型` 由你测试后填写。
- `是否正确`：预测路由等于真实路由填 `1`，否则填 `0`。
- `Accuracy = 路由正确数 / 总样本数`。
- `某类别 Recall = 该类别识别正确数 / 该类别真实样本数`。
- QA 检索评测建议只看检索阶段 top5，不要先看最终生成答案，避免答案生成掩盖检索问题。
